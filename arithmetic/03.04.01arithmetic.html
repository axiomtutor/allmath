Section 04: Sequences and series

Terms: Formula that defines a sequence, common difference, arithmetic sequence, series, index, geometric sequence, common ratio,

Question: The sequence of numbers 4, 7, 10, 13, ... seems like a natural thing to think about. It's a fairly simple sequence and you could imagine it describes, I don't know, taking steps of size 3 over and over--that probably comes up in Science somehow right?

Anyway, what can we say or learn about this sequence? For instance, can we find out what the 100th item is without counting them out?

Answer: Well, it makes some sense that we could multiply 100*3 and add it to 4 to get, if not there, at least close. We can tell the strategy is going to be a little off, though, because if we used it to predict the 4th element we would get 4+4*3=16 and yet we know the right answer for the 4th element is 13. Clearly we need to multiply by one fewer because we don't actually add anything for the first term. I.e. to get the 4th element we find 4+3*3 = 13 and to get the 100th element we would compute 4+99*3 = 301.

In general we can summarize this finding by saying that the nth element is 4+3*(n-1). This is a formula that defines the sequence. Obviously things change if we pick a different starting point and a different step-size. We will denote the first element as a1 and the step size as d. Therefore in this example a1 = 4 and d = 3.

Exercise: Find a formula that defines the sequence that starts at a1 = 5 and uses step-size of d = _2. Use it to find the 100th element.

Answer: 5-2(n-1) is the formula. The 100th element is 5-2(100-1) whatever that is.

We call the "step-size" the more official-sounding phrase common difference. Hence in this problem the common difference was _2 and in the question before, it was 3. If you take any two adjacent elements in the sequence you will get the common difference.

Exercise: Any sequence that can be defined by a formula like this is called an arithmetic sequence. In order to talk about the elements in a sequence we will use the notation a1, a2, and a3 to express the first, second, and third elements of the sequence. So in the first question we had a1 = 4, a2 = 7, and a3 = 10. In the previous exercise we had a1 = 5, a2 = 3, and so on.

I emphasize that the subscripts don't really "mean" anything! Many high school, and some college students misunderstand this, thinking that the subscript 1 maybe means that the number 1 is used in computation. IT'S NOT. It's just keeping track of which is the first element of the sequence. We will use this notation for all sequences, not just arithmetic ones.

Exercise: Find a50 in an arithmetic sequence with a10 = 10 and a11 = 22. Hint: Find the common difference using the two known elements of the sequence. You know that they're separated by four other sequence terms.

Answer: The common difference is clearly 22-10 = 12. We need to know what the first term of the sequence is, but we know that a10 = 22 = a1 + 12*11 and therefore a1 = 22-12*11 = _110. Now that we know the initial term and the common difference, it is a direct application of the formula to find the 50th element.

a50 = _110 + 12(49) = 478

Question: What if we sum all of the sequence terms as we go? So for instance if you use the sequence 1, 2, 3, 4, 5, ..., if you take the sums as you go, 1, 3, 6, 10, 15, ... can we define this sequence by a formula as successfully as we did before?

Answer: You might see that the sequence grows faster than any arithmetic sequence does, and may employ a quadratic term, i.e. n^2. This answer gets the first number right, 1^2 = 1. However it gets the second number wrong, 2^2 = 4 where we want to get 3. The answer gets worse on the next element, 3^2 = 9 where we want 6. Because our answer starts close to right but gets worse and worse, perhaps if we subtract n we will get the correction--n grows kind of like our error does, maybe they'll cancel each other.

n^2-n is clearly wrong because it starts at 0 when n=1. A better next guess is n^2 - n + 1. This generates the sequence 1, 3, 7, 13, ... Closer but still growing too fast! If this is growing too fast but an arithmetic sequence is too slow, maybe we can divide it by 2. To get the first term right that seems to require `\frac{n^{\wedge 2} + n}{2}` because with this if you use n=1 you get out the right value, 1. This formula generates the sequence 1, 3, 6, 10, 15, ... It seems we got it right!

That was a lot of guess-and-check, though. Wasn't there a better way to obtain this result? Yep! A grid of dots can make the point. Represent 1+2+3+4+5 in the following way.

__Example__

To sum these easily, especially in terms of multiplication, it would be nice to find the area of the rectangle of dots, so to speak. To try to fill that in, we can fill the spaces with blue dots.

__Example__

Now the area of dots is clearly n(n+1) but the ones we want are just half of this! Therefore the formula is `\frac{n(n+1)}{2}`, which when you multiply it out, gets you the same formula we found by guessing!

Exercise: To express a sum over a sequence, we use the `\sum` notation. This is called a series. More precisely, we will write

$$\sum_{i=1}^5a_i$$

to express a1+a2+a3+a4+a5. Notice that the bottom of the `\sum` symbol tells you where to start, and the top tells you where to end. The expression in front tells you what you're adding. The i we call the index of the series. Let's suppose that we use two very random sequences, not generated by any formula that I know of. a1=2, a2=_1, a3=5 and the other sequence b1=_2, b2=_1, b3=1. Then we can form the sum

$$\sum_{i=1}^3a_i b_i$$

We sequentially replace i with the values from 1 to 3 and then sum at the end. We get (2)(_2) + (_1)(_1) + (5)(1) = _4 + 1 + 5 = 2.

We could alternatively have formed

$$\sum_{i=1}^2a_i^{\wedge b_i}$$

which is 2^_2 + (_1)^_1 = 1/4 - 1 = _3/4.

So the exercise is to find

$$\sum_{i=1}^3 a_i/b_i$$

Answer: 2/_2 + _1/_1 = _1+1 = 0.

Exercise: We also use this notation, not just for random sequences, but also sequences determined by a formula. If an = 2+3(n-1) and `b_n = \frac{n(n+1)}{2}` then we can form the sum

$$\sum_{i=1}^3 a_ib_i$$

and infer that this is (2)(1) + (5)(3) + (8)(6) = 2 + 15 + 48 = 65.

Your exercise is to use these sequences to find

$$\sum_{i=1}^3 a_i$$

and

$$\sum_{i=1}^2 a_i-b_1$$

Answer: For the first one, it's 2+5 = 7.

For the second, I'll use parentheses to separate the terms of the series. (2-1) + (5-3) = 3

Exercise: There is no need to use ai inside the summation. We can put the formula in there directly. Here are some examples.

$$\sum_{i=1}^4i = 1+2+3+4$$

$$\sum_{i=3}^6i^{\wedge 2} = 3^{\wedge 2} + 4^{\wedge 2} + 5^{\wedge 2} $$

$$\sum_{i=2}^5 \log(2,i-1) = \log(2,1) + \log(2, 2) + \log(2,3) + \log(2,4)$$

How can we use this notation to find the formula that we found for 1+2+3+4?

Answer: If we just sum the first four numbers,

$$\sum_{i=1}^4 i = \frac{4(4+1)}{2}$$

If we sum up to an arbitrary n it's

$$\sum_{i=1}^n i = \frac{n(n+1)}{2}$$

Exercise: To practice the use of the `\sum` notation, suppose that a1=20, a2=24, a3=22, a4=18. Use the `\sum` notation to express the average of these numbers, i.e. the sum of all the elements, divided by the number of elements.

Answer:

$$\frac{\sum_{i=1}^4a_i}{n}$$

Exercise: Suppose ai and bi are the coordinates of a row and column matrix, respectively. For concreteness, suppose v = [2, 4, -1, 0] and w = [1, 2, _1, _2]t. Then we'd have a1=2 and a2=4 and so on. Likewise b1=1 and so on again. Use the `\sum` notation to express the dot-product v•w.

Answer:

$$v\cdot w = \sum_{i=1}^4a_ib_i$$

Exercise: The `\prod` symbol means repeated multiplication in the way that `\sum` means repeated addition. Use the `\prod` notation to express the factorial.

Answer:

$$n! = \prod_{i=1}^n i$$

Exercise: There is actually yet another way of deriving this formula, this one sometimes attributed to the mathematician Gauss. It is to take the sum 1+2+3+...+98+99+100, for instance, then you can re-arrange the sum. Putting 1 together with 100, and putting 2 together with 99, and so on, you always get 101. Can you complete the derivation?

Answer: There are 100 numbers here, so there are 50 pairs, all of them summing to 101, therefore the resulting sum is equal to 50*101 = 5050.

What if there aren't an even number of elements of the series? What if we sum up to 101? An obvious fix is just to sum up to 100 and then add the 101 at the end.

If we want to prove that the formula is true through this argument, we can say that if n is even then 1+2+...+(n-2)+(n-1)+n = (1+n) + (2+n-1) + ... where every term evaluates to 1+n and there are n/2 terms. Thus the sum is (1+n)(n/2) which is the same formula as before. If n is odd then you use 1+2+3+...+(n-1)+n = (1+n-1) + (2+n-2) + ... where there are (n-1)/2 terms each evaluating to n and again the same formula results.

Question: Now that we know how to sum 1+2+3+4+... How can we sum other arithmetic sequences? What is the formula for 4+7+10+...?

Answer: We can represent this sum as

$$\sum_{i=1}^n(4+3(i-1))$$

and we will use some properties of a series, but first it'll be better to simplify

$$\sum_{i=1}^n(1+3i)$$

Now the first series property we will use is that the sum is distributive over addition.

$$\sum_{i=1}^n(1) + \sum_{i=1}^n (3i)$$

The left-hand summand here is just adding 1 over and over n times. Of course that's just going to be equal to n! In the right-hand summand, we can use the property of series that a constant factor (i.e. a factor with no i in it) can factor out of a sum. Thus this transforms into

$$n + 3 \sum_{i=1}^n i$$

At this point we can use the formula that we've found before.

$$n+3\left(\frac{n(n+1)}{2}\right)$$

I call this last formula the summing the index formula. For instance this can be used to tell us 4+7+10+13+16 is the same as `\sum_{i=1}^5(4+3(i-1)) = 5+3\left(\frac{5(5+1)}{2}\right) = 50`.

Exercise: Find the sum of the first 100 terms of 4+7+10+...

Answer:

$$\sum_{i=1}^100 (4+3(n-i)) = 100+3(100*101)/2 = 100+350*101 = 35450$$

Exercise: Find a formula for 5 + 2 + _1 + _4 + ...

Answer:

The first equality below is just due to simplification. The second is due to distribution. The third is due to factoring in both series. The fourth is summing 1 (left series) and summing the index (right series)

$$\sum_{i=1}^n (5+_3(i-1)) = \sum_{i=1}^n(8-3i) = \sum_{i=1}^n8 + \sum_{i=1}^n\_3i = $$

$$8\sum_{i=1}^n1 - 3\sum_{i=1}^ni = 8n - 3\left(\frac{n(n+1)}{2}\right)$$

Exercise: Use the formula found before to sum the 5th to the 100th elements of the sequence.

Answer: Probably the easiest way to do this is to first figure out the sum of the first 100 elements.

$$8(100) - 3\left(\frac{100(100+1)}{2}\right)$$

This has too much in it, but we can subtract it out. The part we want to subtract out is

$$8(5) - 3\left(\frac{5(5+1)}{2}\right)$$

I won't bother to subtract these for you, but the difference of these two expressions is the answer.

Question: Having studied arithmetic sequences, are there other interesting sequences? What if, rather than use repeated addition, we use repeated multiplication?

Answer: This is a geometric sequence. Here's an example, 6, 12, 24, 48, 96, ... The formula for an arithmetic sequence used multiplication, a1 + d(n-1). It's not hard to expect that the formula for the geometric sequence would use exponentiation. It's a1 r^n-1. As usual a1 expresses the first term of the sequence, like 6 in the example above. r is called the common ratio. If you divide any two adjacent terms by each other, you get the common ratio: 12/6 = 2.

We will only study these two sorts of sequences, so we can summarize the sequence formulae now.

Arithmetic: an = a1 + d(n-1)

Geometric: an = a1 r^n-1

Exercise: Express the 100th term of the sequence 6, 12, 24, ... but don't bother to evaluate it--the number is too large.

Answer: 6(2^99)

Exercise: Find the initial value and common ratio of the sequence determined by `\frac{1.3^{\wedge n-1}}{2}`.

Answer: The initial value is 1/2 and the common ratio is 1.3.

Question: Can we sum a geometric series? In a good (efficient) way?

Answer: Of course the bad, brute-force way, 6+12+24 = 42 is possible. But the smarter way involves some serious trickery. Let's try to find 6+12+24+48+96+192+384 efficiently. First we can notice that if you multiply everything by 2 you will shift most of the terms over.

2( 6+12+24+48+96+192+384 ) = 12+24+48+96+192+384+768

Now what we can infer is that if you take the difference

12+24+48+96+192+384+768

- (6+12+24+48+96+192+384)

and most of the terms cancel except 768 and _6. These sum to 762.

Now, if we call the s = 6+12+24+48+96+192+384 effectively what we just showed was that 2s - s = 762. If you know Algebra you know that the left-hand side simplifies to s but since I want to try not to reply on Algebra, let's just say that 2s-s = s+s-s = s.

Putting this all together we've just shown that s=762.

If we took the sequence 1 + 1/3 + 1/9 + 1/27 + 1/81 then we have a geometric sequence with initial value 1 and common ratio 1/3. If we reproduce the reasoning that we just used, we would argue that

s = 1 + 1/3 + 1/9 + 1/27 + 1/81

(1/3)s = 1/3 + 1/9 + 1/27 + 1/81 + 1/243

s-(1/3)s = 1-1/243 = 242/243

Now this is not yet the actual answer, because it's really s - (1/3)s. At this point Algebra is somewhat unavoidable--I'll keep it to a minimum. Here we can factor out s and get s(1-1/3) which is s(2/3). So we have

(2/3)s = 242/243

To get s we can divide by 2/3 and get

$$s = \frac{242/243 }{2/3}$$

and you can compute it further if you like, but I'll leave it here because the point has been made.

To put these results in the `\sum` notation we can write

$$\sum_{i=1}^76(2^{\wedge i-1}) = 762$$

and

$$\sum_{i=1}^41(1/3)^{\wedge i-1} = \frac{242/243}{2/3}$$

Exercise: Find the general formula for the sum of a geometric sequence. That is to say, use the trick above to find a formula for

$$\sum_{i=1}^n a_1 r^{\wedge i-1}$$

Answer: We call this s and multiply by r so that

$$s = a_1 + ra_1 + r^{\wedge 2}a_1 + ... + r^{\wedge n-1}a_1$$

$$rs = ra_1 + r^{\wedge 2}a_1 + ... + r^{\wedge n}a_1$$

and therefore

$$s-sr = a_1 - r^{\wedge n}a_1$$

$$s(1-r) = a_1(1 - r^{\wedge n})$$

$$s = a_1\left(\frac{1-r^n}{1-r}\right)$$

Now we can summarize both the arithmetic series formula and the geometric series formula

Arithmetic: `\sum_{i=1}^na_i = \frac{n(n+1)}{2}`

Geometric: `\sum_{i=1}^n a_i = a_1 ( \frac{1-r^{\wedge n}}{1-r})`

Exercise: Here is a famous connection to a philosophical problem. An arrow is fired toward a target that is, say, 2 meters away. In order to cross that distance, the arrow must first cross half the distance, 1 meter. However, in order to cross that 1 meter, it must cross half of 1 meter, or 1/2 meters. In order to cross that, it must first cross 1/4 meters, and so on. As this process can proceed infinitely, the arrow must number actually move. What resolves this (called Zeno's Paradox)?

Answer: The sum of all of the distances is

$$\sum_{i=1}^\infty 1(1/2)^{\wedge n-1}$$

We know by the formula that this is

$$\left(\frac{1-(1/2)^{\wedge \infty}}{1-1/2}\right)$$

Now of course this is not well defined yet, because it's an infinite sum and it turns into an infinite power. But think of it this way: Every extra power of 1/2 makes it smaller and smaller until, so to speak, at infinity the value of (1/2)^∞ crushes down to 0. Then this whole thing simplifies to

$$\left(\frac{1-0}{1-1/2}\right) = 2$$

The point is that, although the arrow passes through an infinity of distances, those distances become vanishingly small. It turns out that, sometimes, by summing an infinite number of positive quantities you actually result in a finite quantity--if the summands grow small enough fast enough.

Mathematicians and physicists like to convince themselves that this resolves Zeno's Paradox, but it does not, it just re-asserts that a finite distance can be segmented into infinitely many "partitions" which was the premise in the first place. This adds nothing but formulae to the analysis of the philosophical problem. Of course the paradox can be resolved, but this is not the way. It is still a cute connection to the mathematical subject that we're studying for independent reasons.
