Chapter 02: Propositional Logic

Section 01: Propositions and operations

Abstract: The basic syntax, how to form well-formed formulas; also the basic semantics, how to evaluate the truth of a proposition.

Terms: atomic proposition, negation, proposition, true, false, structure, negand, conjunction, conjunct, main operator, truth-table, equivalent, double-negation, standard truth-table

Symbols: 𝕡, 𝕢, 𝕣, ¬, T, F, 𝓜, ∧

Question: What will be our basic unit of study?

Answer: The proposition. Atomic propositions are sentences that are either true or false. For instance "One plus one is two" is a true proposition and "Russia is small" is a false proposition. The sentence "Which way did he go?" is a well-formed sentence and all, but it's neither true nor false and therefore not a proposition; nor are exclamations (sometimes), nor are commands.

We will not be interested in any particular proposition, but rather the relationships that certain kinds of propositions have to other propositions. Therefore, in order to ignore the details of a given proposition, we use the symbol 𝕡 to denote any proposition. If we want to talk about a proposition besides 𝕡 then we may use 𝕢 or 𝕣, and if we need more than these we may use subscripts to index an infinite set of propositions, e.g. 𝕡2.

Question: What do we do with them?

Answer: These propositions are intended to be simple, and we build up more complex propositions using what are called logical operations. The first logical operation to consider is negation, which represents the denial of a proposition, like "One plus one is not two." We use the ¬ symbol to denote negation, so that if 𝕡 represents "one plus one is two" then ¬𝕡 represents "one plus one is not two". Although 𝕡 is an atomic proposition, ¬𝕡 is not. Rather, we call ¬𝕡 a complex proposition, or just a proposition for short.

Although we want to keep syntax and semantics separate, it is most helpful to introduce them in tandem. I've already said what the intuitive semantics of this symbol are, the natural language "negation" or "not". However, in Propositional Logic what we mean by "semantics" is the truth of a proposition, or how the symbol affects the truth of a proposition. We will use the symbols T and F to denote the value true and the value false. It is up to Science, or Philosophy, or some other discipline to decide whether a proposition gets the value true or false; logicians will consider all possibilities and remain agnostic about the truth or falsity of any given atomic proposition. We only concern our selves with: if 𝕡 is true then ... or if 𝕡 is false then ...

Now, when 𝕡 is true ¬𝕡 is false, naturally; and vice versa. We can also form more complicated propositions, although I won't promise that they're terribly interesting. ¬ ¬ 𝕡 is another sentence we're able to form with the symbols that we have so far. What happens if 𝕡 is true? In that case the first negation "toggles" it to false and then the next negation toggles it back to true; likewise if 𝕡 is false then ¬ ¬ 𝕡 is false.

Whenever we hypothesize a truth-value for 𝕡 we are considering what is called a structure. A structure is an assignment of truth-values to the atomic propositions in a language. We will denote a structure by 𝓜 and we will write 𝕡𝓜 = T to express that the structure assigns 𝕡 true, and 𝕡𝓜 = F to of course denote that it's false. When we select a structure, we are interested in using it to decide the truth-value of complex propositions. We've seen that, if 𝕡𝓜 = T then (¬𝕡) 𝓜 = F; and when 𝕡𝓜 = F then (¬𝕡) 𝓜 = T. Question: Ok, negation. What else you got?

Answer: Another common construction is conjunction, which in English most often is expressed with "and". Take the sentence "1+1=2 and 1+2=3". The claim made by that proposition is that both component sentences are true--so if one (or both) of them is false, the claim is false. For this we use the symbol ∧ so that, if 𝕡 represents the proposition "Russia is small" and 𝕢 represents "Costa Rica is small" then 𝕡∧𝕢 represents "Russia and Costa Rica are small". Notice that I could have chosen to say this as "Russia is small and Costa Rica is small" but that's unnecessarily verbose. When "translating" sentences from symbols to natural language, it's best to choose the most natural possible language.

Also notice that propositions like "Russia is large" may be considered inadequately vague to be an acceptable proposition. Large compared to what, people? Then yeah, of course. Large compared to stars? No, actually quite small. Again, we as logicians are not in the truth-assignment business. We can decide in private whether the sentence is true, or we can hash it out with others--but the logician only takes over once the assignments to the atoms has already been made.

In the language of structures we have, (𝕡∧𝕢)𝓜 = T if and only if both 𝕡𝓜 = T and 𝕢𝓜 = T. Thus when 𝕡𝓜 = T but 𝕢𝓜 = F we get the result that (𝕡∧𝕢)𝓜 = F. This may seem unfair, like the sentence deserves half-truth or something, when one of the sentences is true but the other false. However, this is not how we choose to interpret the meaning of conjunction or the notion of truth.

As a point of vocabulary, we call the component sentences in a conjunction a conjunct.

Question: Which rule shall we pick to disambiguate negation and conjunction?

Answer: The standard treatment of this issue is to make negation take the smallest scope possible, that is to say, we assume ¬𝕡∧𝕢 means (¬𝕡)∧𝕢.
